# Natalia's Therapy Space

A web application for therapy space management built with Vite, React, TypeScript, and shadcn-ui.

## Installation

To install the necessary dependencies, run:

```sh
npm install
```

## Running the Application

To start the development server:

```sh
npm run dev
```

## Deployment

To build the project for production:

```sh
npm run build
```

This will create a `dist` folder that you can deploy to any static hosting service like Vercel, Netlify, or GitHub Pages.
